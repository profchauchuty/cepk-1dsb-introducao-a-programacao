# Algoritmo

## 1. O que é um Algoritmo
- Sequência finita de passos lógicos para resolver um problema ou executar uma tarefa.
- Algoritmo é um passo a passo organizado e estruturado.

## 2. Algoritmos no Cotidiano
- Receitas de bolo  
- Manual de montagem de móveis  
- Instruções de caixa eletrônico  
- Regras para cálculo de média escolar  

## 3. Exemplo em Linguagem Natural
**Problema:** Fazer café  
**Algoritmo:**  
1. Colocar água na chaleira  
2. Ligar o fogão  
3. Esperar a água ferver  
4. Colocar pó de café no filtro  
5. Despejar a água quente no filtro  
6. Servir o café  

## 4. Características de um Bom Algoritmo
- Finito: possui início e fim  
- Claro: não gera dúvidas  
- Ordenado: passos em sequência lógica  
- Executável: possível de realizar  
- Correto: resolve o problema proposto  

## 5. Exemplo Prático
**Problema:** Calcular a média de duas notas  
**Algoritmo:**  
1. Receber a primeira nota  
2. Receber a segunda nota  
3. Somar as notas  
4. Dividir o resultado por 2  
5. Mostrar a média  

**Teste de mesa:**  
- Nota 1 = 8, Nota 2 = 6  
- Soma = 14, Média = 7
