# Exercícios: Funções

## Objetivo
Criar programas simples em Portugol utilizando **FUNÇÕES** para modularizar, organizar e reutilizar códigos.

---

# Exercícios com Funções

## 1. Função de Boas-Vindas Personalizada
- Criar uma função chamada `boasVindas`.
- A função deve receber o nome de uma pessoa.
- Exibir:
  - "Seja bem-vindo(a), NOME".

---

## 2. Função para Calcular Dobro
- Criar uma função chamada `dobro`.
- A função deve receber um número.
- Retornar o dobro desse número.
- Exibir o resultado no programa principal.

---

## 3. Função para Calcular Área do Retângulo
- Criar uma função chamada `calcularArea`.
- A função deve receber:
  - Base.
  - Altura.
- Retornar a área do retângulo.
- Exibir o resultado no programa principal.

---

## 4. Função para Converter Metros em Centímetros
- Criar uma função chamada `converterCentimetros`.
- A função deve receber um valor em metros.
- Retornar o valor convertido para centímetros.
- Exibir o resultado.

Exemplo:

```text
2 metros = 200 centímetros
```

---

## 5. Função para Verificar Maioridade
- Criar uma função chamada `isMaiorDeIdade`.
- A função deve receber uma idade.
- Retornar:
  - Verdadeiro se a idade for maior ou igual a 18.
  - Falso caso contrário.
- Exibir o resultado.

---

## 6. Função para Calcular Desconto
- Criar uma função chamada `calcularDesconto`.
- A função deve receber:
  - Valor do produto.
  - Percentual de desconto.
- Retornar o valor final do produto.
- Exibir o resultado.

---

## 7. Função para Calcular IMC
- Criar uma função chamada `calcularIMC`.
- A função deve receber:
  - Peso.
  - Altura.
- Retornar o valor do IMC.
- Exibir o resultado.

Fórmula:

```text
IMC = peso / (altura²)
```

---

## 8. Função para Verificar Aprovação
- Criar uma função chamada `verificarAprovacao`.
- A função deve receber a média final de um aluno.
- Retornar:
  - "Aprovado" se a média for maior ou igual a 6.
  - "Reprovado" caso contrário.
- Exibir o resultado.

---

## 9. Função para Calcular Potência
- Criar uma função chamada `potencia`.
- A função deve receber:
  - Base.
  - Expoente.
- Retornar o resultado da potência.
- Exibir o resultado.

Exemplo:

```text
2 elevado a 3 = 8
```

## 10. Sistema de Calculadora com Funções

- Criar um sistema de calculadora utilizando funções.

Menu:

```text
1 - Somar
2 - Subtrair
3 - Multiplicar
4 - Dividir
5 - Sair
```

Regras:
- Enquanto o usuário não escolher a opção `5`:
  - Mostrar o menu.
  - Ler a opção escolhida.

### Função Somar
- Criar uma função chamada `somar`.
- Receber dois números.
- Retornar a soma.

### Função Subtrair
- Criar uma função chamada `subtrair`.
- Receber dois números.
- Retornar a diferença.

### Função Multiplicar
- Criar uma função chamada `multiplicar`.
- Receber dois números.
- Retornar o produto.

### Função Dividir
- Criar uma função chamada `dividir`.
- Receber dois números.
- Verificar:
  - Se o divisor for diferente de zero:
    - Retornar o resultado da divisão.
  - Senão:
    - Mostrar:
      - "Não é possível dividir por zero".

### Opção 5 — Sair
- Mostrar:
  - "Calculadora encerrada".

### Qualquer Outra Opção
- Mostrar:
  - "Opção inválida".
